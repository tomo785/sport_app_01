# 任务流程图界面 - 开发文档
## 功能概述
新增任务流程图界面，实现目标分解为多步骤任务链的可视化展示，支持任务卡片拖拽排序、优先级标记、选中高亮等交互功能。
## 界面特性
### 1. 自定义导航栏
- 不占用系统导航栏，完全自定义
- 顶部状态栏自动适配（适配刘海屏）
- 标题"任务卡片"居左显示
- 右侧进度徽章显示完成数量
- 背景毛玻璃效果
### 2. 训练计划切换
- 支持两种训练计划切换：
  - **力量训练**：杠铃深蹲、哑铃卧推等传统力量动作
  - **5km跑步**：完整的跑步训练流程
- 点击标签即可切换，带动画过渡效果
### 3. 5km跑步训练计划
包含以下任务链：
1. **热身慢跑** (1km) - 轻松慢跑，逐渐提升心率
2. **动态拉伸** (5分钟) - 高抬腿、踢臀跑、开合跳
3. **800米间歇跑** (3.2km, 4组) - 快跑800m + 慢跑400m恢复
4. **冷身慢跑** (0.8km) - 慢速恢复跑
5. **结束拉伸** (6分钟) - 腿部拉伸、髋部放松
### 4. 补水提醒卡片
- 青蓝色渐变卡片设计
- 实时显示已饮水量
- 点击打开补水记录弹窗：
  - 今日已饮水统计
  - 目标进度条（800ml）
  - 快捷添加按钮：+150ml / +250ml / +500ml
  - 重置功能
- 仅在跑步计划下显示
### 2. 日历式日期选择器
- **默认显示本周**: 只展示当前周的7天（周日-周六）
- **本周范围标题**: 显示本周日期范围（如：4月6日-4月12日）
- **星期标题**: 日一二三四五六，周末橙色标识
- **本周日期**:
  - 今天蓝色高亮背景
  - 选中日期蓝紫渐变背景
  - 有任务的日期显示小圆点
  - 周末日期橙色
- **展开功能**（点击▼按钮）:
  - 显示完整月历（整月日期）
  - 显示月份选择器（12个月份网格）
  - 年份切换（左右箭头）
  - 点击月份跳转至该月
### 3. 任务链
- **垂直流程连线**: 左侧虚线连接各任务节点
- **任务卡片**:
  - 右上角优先级标签（高/中/低）
  - 左侧拖拽手柄支持长按拖拽
  - 任务图标（根据类型显示不同图标）
  - 任务名称、时长
  - 组数、消耗卡路里详情
  - 状态指示器（未开始/进行中/已完成）
  - 已完成任务有绿色遮罩和check图标
- **顺序序号**: 左侧圆圈内显示任务顺序
### 4. 选中高亮
- 选中卡片边框变为蓝色
- 轻微放大效果
- 脉冲动画提示
### 5. 底部操作栏
- 显示选中任务名称和状态
- 动态按钮（开始/继续/查看）
- 渐变色彩设计
### 6. 结束节点
- 橙黄渐变卡片
- 奖杯图标
- 表示今日训练完成
## 设计规范
### 颜色
- 背景: `linear-gradient(180deg, #f8fafc 0%, #f1f5f9 100%)`
- 主色: `#3b82f6` (蓝色)
- 成功: `#10b981` (绿色)
- 警告: `#f59e0b` (橙色)
- 优先级高: `#dc2626` (红色)
- 优先级中: `#d97706` (黄色)
- 优先级低: `#059669` (绿色)
### 圆角
- 卡片: 20-24px
- 按钮: 20-36px
- 图标容器: 14-18px
- 标签: 10-12px
### 阴影
- 卡片: `0 3px 16px rgba(0, 0, 0, 0.05)`
- 选中: `0 6px 24px rgba(59, 130, 246, 0.2)`

## 使用方式
### 1. 查看界面效果
直接打开 `task-flow-ui.html` 文件，可在浏览器中预览移动端界面效果。
### 2. 访问方式
任务流程图已作为独立的底部导航栏页面：
- 点击底部导航栏的 "**流程**" 选项卡即可进入
### 3. 直接访问（编程式导航）
```javascript
uni.switchTab({
  url: '/pages/task/task-flow'
})
```
## API 接口
### 新增接口
| 接口 | 方法 | 说明 |
|------|------|------|
| /task/flow | GET | 获取任务流程图数据 |
| /task/flow/reorder | POST | 更新任务顺序 |
| /task/{id}/priority | POST | 更新任务优先级 |
| /task/{id}/start | POST | 开始指定任务 |
| /task/batch/status | POST | 批量更新任务状态 |

## 数据结构
### 任务对象
```typescript
interface Task {
  id: number           // 任务ID
  name: string         // 任务名称
  type: string         // 类型: warmup/strength/cardio/core/stretch/rest
  priority: string     // 优先级: high/medium/low
  status: number       // 状态: 0-未开始, 1-进行中, 2-已完成
  duration: number     // 预计时长(分钟)
  sets: number         // 组数
  calories: number     // 消耗卡路里
  order: number        // 顺序
}
```
### 目标对象
```typescript
interface Goal {
  id: number           // 目标ID
  title: string        // 目标标题
  description: string  // 目标描述
  targetProgress: number  // 目标进度值
  currentProgress: number // 当前进度值
}
```
## 交互说明
### 拖拽排序
1. 长按任务卡片左侧的手柄
2. 上下拖动调整位置
3. 松开手指完成排序
4. 自动调用API保存新顺序
### 选择任务
1. 点击任意任务卡片
2. 卡片边框变为蓝色，有脉冲动画
3. 底部操作栏显示该任务信息
4. 按钮根据任务状态变化（开始/继续/查看）
### 切换日期
1. 点击日期选择器的日期项
2. 重新加载该日期的任务流程
3. 任务链动态更新
