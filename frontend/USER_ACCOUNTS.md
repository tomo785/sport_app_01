##  同步到数据库
### 方式一：SQL 脚本（推荐）
```bash
# 登录 MySQL
mysql -u root -p
# 执行用户初始化脚本
source scripts/init-user.sql
```
### 方式二：Node.js 脚本
```bash
# 安装依赖
npm install bcryptjs mysql2
# 修改数据库配置（编辑 scripts/init-user.js）
# 运行脚本
node scripts/init-user.js
```
### 方式三：同时创建管理员和普通用户
```bash
# 管理员
node scripts/init-admin.js
# 普通用户
node scripts/init-user.js
```
