# 功能模块清单

## 1. 用户模块

| 功能 | 说明 | 优先级 |
|------|------|--------|
| 手机号注册/登录 | 手机号+短信验证码登录 | P0 |
| 微信登录 | 微信授权登录 | P1 |
| 用户信息完善 | 昵称、头像、性别、年龄、身高、体重 | P0 |
| 个人中心 | 展示用户基本信息和运动概况 | P0 |
| 退出登录 | 清除本地token和用户信息 | P0 |

---

## 2. 运动记录模块

| 功能 | 说明 | 优先级 |
|------|------|--------|
| 运动类型选择 | 跑步、健走、骑行三种类型 | P0 |
| 实时定位追踪 | 获取GPS位置，记录运动轨迹 | P0 |
| 实时数据展示 | 速度、距离、时长、卡路里实时更新 | P0 |
| 开始/暂停/继续/结束 | 控制运动状态 | P0 |
| 轨迹记录 | 按时间间隔记录位置点并上传 | P0 |
| 运动详情查看 | 查看历史运动记录的完整轨迹和详细数据 | P0 |
| 运动列表 | 分页展示历史运动记录 | P0 |
| 删除运动记录 | 删除不需要的运动记录 | P1 |

---

## 3. 数据统计模块

| 功能 | 说明 | 优先级 |
|------|------|--------|
| 日报表 | 当日运动次数、距离、时长、卡路里统计 | P0 |
| 周报表 | 本周运动数据汇总，支持周对比 | P0 |
| 月报表 | 本月运动数据汇总，支持月对比 | P0 |
| 历史趋势 | 运动数据的时间趋势图表展示 | P1 |
| 运动类型分析 | 各运动类型的数据占比分析 | P1 |
| 累计统计 | 总运动次数、总距离、总时长等累计数据 | P0 |

---

## 4. 目标设置模块

| 功能 | 说明 | 优先级 |
|------|------|--------|
| 创建目标 | 设置距离、时长、卡路里、次数等类型的目标 | P0 |
| 目标列表 | 查看用户设置的所有目标 | P0 |
| 目标进度 | 显示目标完成百分比和进度条 | P0 |
| 编辑目标 | 修改目标值、类型和周期 | P0 |
| 删除目标 | 删除不需要的目标 | P0 |
| 目标周期 | 支持每日、每周、每月目标 | P0 |
| 目标提醒 | 目标达成时的通知提醒 | P1 |

---

## 5. 地图模块

| 功能 | 说明 | 优先级 |
|------|------|--------|
| 实时轨迹显示 | 运动过程中实时显示运动轨迹 | P0 |
| 轨迹回放 | 查看历史运动轨迹的动态回放 | P0 |
| 轨迹分享 | 将运动轨迹分享到社交平台 | P2 |
| 常用路线 | 保存和复用常用运动路线 | P2 |
| 地图样式 | 支持标准、卫星、夜视等多种地图样式 | P1 |

---

## 6. 设置模块

| 功能 | 说明 | 优先级 |
|------|------|--------|
| 账号设置 | 修改密码、绑定/解绑微信 | P1 |
| 隐私设置 | 运动记录可见性、数据分享设置 | P1 |
| 通知设置 | 目标提醒、运动提醒等通知开关 | P1 |
| 单位设置 | 公制/英制单位切换（km/mile） | P1 |
| 语音播报 | 实时数据语音播报设置 | P1 |
| 关于App | 版本信息、隐私政策、用户协议 | P0 |
| 意见反馈 | 用户意见反馈功能 | P2 |

---

## 功能优先级说明

| 优先级 | 说明 |
|--------|------|
| P0 | 核心功能，MVP版本必须实现 |
| P1 | 重要功能，V1.0版本实现 |
| P2 | 辅助功能，后续版本迭代 |

---

## 模块依赖关系

```
用户模块 (基础)
    ├── 运动记录模块 (核心)
    │   ├── 数据统计模块 (依赖运动记录)
    │   ├── 目标设置模块 (依赖运动记录和统计)
    │   └── 地图模块 (依赖运动记录)
    └── 设置模块 (独立)
```

---

## 用户核心流程

### 运动记录流程
```
登录 → 选择运动类型 → 开始运动 → 实时记录轨迹 → 结束运动 → 查看详情
```

### 目标设置流程
```
登录 → 进入目标页面 → 创建目标 → 选择类型和周期 → 设置目标值 → 保存
```

### 数据统计流程
```
登录 → 进入统计页面 → 选择时间范围 → 查看报表/图表
```
